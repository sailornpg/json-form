# Triage Labels

这些 skill 使用五个规范化的分诊角色。本文件将这些角色映射到本仓库 issue tracker 中实际使用的标签字符串。

| Label in mattpocock/skills | Label in our tracker | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者先评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 需求已明确，AFK agent 可直接接手 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不计划处理 |

当某个 skill 提到某个分诊角色时，使用上表右侧对应的标签字符串。
